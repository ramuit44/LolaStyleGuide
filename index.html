Bootstrap

LOLA is built on the Twitter Bootstrap framework v3.0

Default Bootstrap theme is used as much as possible, with some custom css classes contained within a 'macStyles.css' file.

For icons "Font Awesome Icons" were used.

The custom CSS file should be included below the default Bootstrap CSS files. Normalize.css is used to reset all css rules before loading the Bootstrap CSS files

Bootstrap.min.js is the supporing file for the bootstrap.css for all toggle and script related actions.

Google's JS API is used to draw charts.

mac.js is used for custom, LOLA-specific java script code

tableSorter.js is used to sort tables, etc.

HTML
<script src="js/jquery.min.js"></script>
<script src="js/bootstrap.min.js"></script>
<script src="js/tableSorter.min.js"></script>
<script src="js/mac.js"></script>
<script src="https://www.google.com/jsapi"></script>
<link href="css/normalize.css" rel="stylesheet" />
<link href="css/bootstrap.min.css" rel="stylesheet" />
<link href='css/font-awesome.css' rel='stylesheet' />
<link href="css/macStyles.css" rel="stylesheet" />

Button navigation and groups

Button navigation

Buttons can be used as navigation elements, as per below.

HTML

<button type="button" class="btn btn-link active">Current link</button>
<button type="button" class="btn btn-link">Next link</button>
<button type="button" class="btn btn-link">Last link</button>

Anchor tags, <a>, can also be used to control 'pill' style navigation.

Current link Next link Last link

HTML

<a href="#" class="btn btn-link active">Current link</a>
<a href="#" class="btn btn-link">Next link</a>
<a href="#" class="btn btn-link">Last link</a>

Button groups

Buttons can also be grouped.

HTML

<div class="btn-group">
	<button type="button" class="btn btn-default">Increase</button>
	<button type="button" class="btn btn-default">Variation</button>
	<button type="button" class="btn btn-default">Extend</button>
</div>

Individual buttons can be disabled at any point

HTML

<div class="btn-group">
	<button type="button" class="btn btn-default disabled">Increase</button>
	<button type="button" class="btn btn-default">Variation</button>
	<button type="button" class="btn btn-default">Extend</button>
</div>

Fonts and Typography

Fonts and text

LOLA uses Helvetivca Neue as the primary font. Where Helvetica Neue is not available, a Google Font, 'Roboto', has been used as a substitute. This is brought into the stylesheet via the @import rule.

CSS

@font-face {
  font-family: 'Roboto';
  font-style: normal;
  font-weight: 400;
  src: local('Roboto Regular'), local('Roboto-Regular'), url(../fonts/google.woff) format('woff');
}
body { font-family: "Helvetica Neue", "Roboto", Helvetica, Arial, san-serif; }

Browser default sizing is used where possible, with default Bootstrap sizing used as a backup where browser sizing is not appropriate or applicable.

    Charts

    Charts are drawn using the Google Charts API found here

    The API library must be included in the header of any HTML page that requires a chart to be drawn and the library can only be accessed via Google - we cannot download and host the API locally.

    HTML
    <script src="https://www.google.com/jsapi"></script>

    An example output of a chart is shown below. The chart is drawn using simple JavaScript. Full documentation of available customisations can be found at the Google Developers site

    Example chart

    HTML

    JavaScript

    
<script type="text/javascript">
  google.load("visualization", "1", {packages:["corechart"]});
  google.setOnLoadCallback(drawChart);
  function drawChart() {

    var data = google.visualization.arrayToDataTable([
      ['Task', 'Hours per Day'],
      ['Work',     11],
      ['Eat',      2],
      ['Commute',  2],
      ['Watch TV', 2],
      ['Sleep',    7]
    ]);

    var options = {
      title: 'My Daily Activities'
    };

    var chart = new google.visualization.PieChart(document.getElementById('piechart')); //ID name should be intuative -												//the ID name given here is an eample

    chart.draw(data, options);
  }
</script>
